## Welcome

Use the GetMiPay API to collect payments, send payouts, check balances, convert currencies, and manage virtual cards.

## Authentication

Authenticate once with your public and private API keys:

```json
{
  "public_apikey": "gmp_pk_...",
  "private_secretkey": "gmp_sk_..."
}
```

The authentication response returns a Tymon JWT bearer token that expires after 24 hours. Send that token on protected requests:

```bash
Authorization: Bearer <jwt>
```

After login, protected API routes should use only the bearer token.
